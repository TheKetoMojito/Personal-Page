SmartWatch Controlled Skateboard

Description:

A motorized skateboard controlled by a Pebble SmartWatch.

Technologies:

C, NodeJS, Microcontrollers

Implementation:
  • A pebble application listens for button presses. On each event, a message is sent to a connected Android application.
  • The Android app relays the data to a NodeJS server hosted on an Intel Edison microcontroller.
  • The Edison interpretes the signal and adjusts electrical output to the motor.
